# SoftGrowTech_SalesDataDashboard 
A sales data dashboard built in Microsoft Excel as part of the SoftGrowTech internship program.
## Features
- Pivot Tables for sales by order date, category, region, states and profit by states.
- Slicers for interactive filtering by category and region
- Pivot Charts for visual data analysis
# Files
-[Superstore Sales-Dashboard.xlsx](https://github.com/user-attachments/files/29941801/Superstore.Sales-Dashboard.xlsx)
## How to use
- Download the Excel file
- Enable macros if prompted
- Use slicers to filter data by category or region
